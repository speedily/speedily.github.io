Theme Description

She Lancer Portfolio is a modern, award-winning WordPress theme designed for freelancers and creative professionals. It features dual color schemes, interactive animations, and a clean, responsive layout. Showcase your work, services, testimonials, and experience with ease.

Theme Features

  • Responsive Design
  • Custom Portfolio Section
  • Services Showcase
  • Testimonials
  • Experience Timeline
  • Contact Section with Google Maps Integration
  • Dual Color Schemes (Classic & Modern)
  • Interactive Animations
  • SEO Optimized
  • Translation Ready

Required Plugins

  • She Lancer Portfolio CPT
    Handles custom post types for portfolio, services, testimonials, and experience.
    File: inc/she-lancer-cpt/she-lancer-cpt.php
  • She Lancer Portfolio Meta
    Handles meta boxes for testimonials.
    File: inc/she-lancer-meta/she-lancer-meta.php
  • She Lancer Portfolio Dummy
    Handles dummy data import/export functionality.
    File: inc/she-lancer-dummy/she-lancer-dummy.php

How to Use

  • Activate the theme via Appearance > Themes
  • Install and activate the required plugins (included in the theme)
  • Import dummy data from Appearance > Theme Settings (optional)
  • Customize your site via Appearance > Customize
  • Add portfolio items, services, testimonials, and experience via their respective menu items in the WordPress dashboard
